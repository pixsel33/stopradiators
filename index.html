Как будет происходить работа по монтажу системы водяного теплого пола?

  • Подготовка основания

    Пол выравнивается перед укладкой пенополистирола (сбиваются наплывы бетонного раствора)

  • Укладка утеплителя

    Укладываем пенополистирол толщиной определенной в расчёте (по умолчанию 100мм). Сверху укладываем подложку для теплого пола с разметкой.

  • Укладка трубы

    Труба укладывается с определенным шагом в расчете (по умолчанию 150мм). Контуры делаются примерно одинаковой длины для лучшей балансировки системы. Прокладываем кабеля от коллектора к местам будущей установки комнатных термостатов.

  • Монтаж коллектора

    В заранее определенном месте устанавливается коллектор. Затем все трубы подключаются к коллектору с помощью специальных присоединительных элементов.

  • Опрессовка и проверка

    Перед выполнением работ по стяжке система опрессовывается воздухом с фиксацией давления (проверка на герметичность).

  • Заливка стяжки

    После того как убедились, что система герметична выполняем цементно-песчанную стяжку.

  • Установка автоматики

    После окончания чистовой отделки устанавливается автоматика для управления системой отопления.

  • Запуск системы в работу

    Когда стяжка высохла, система запускается в работу с постепенным поднятием температуры.

Установленный в каждой отдельно регулируемой зоне комнатный терморегулятор "понимая", что достигнута заданная температура посылает сигнал на сервомоторы установленные на контурах уложенных в данной зоне. Сервопривод открывает либо закрывает подачу теплоносителя в контур 2.

Регулирование по таймеру

На установленных в отдельно регулируемых зонах термостатических регуляторах есть возможность выставлять необходимую температуру по таймеру, что приводит к дополнительной экономии. В моменты когда Вы отдыхаете ночью либо находитесь в отъезде, система снижает температуру по заданному графику.
